# 🏥 Hospital Appointment Management System

A full-stack web application for managing hospital appointments, 
patients, and doctors built using Java Spring Boot and MySQL.

## 🚀 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript, Bootstrap 5 |
| Backend | Java, Spring Boot 3.x |
| Database | MySQL |
| API | REST API |
| ORM | Spring Data JPA |
| Build Tool | Maven |

## ✨ Features

- 🔐 Admin Login & Authentication
- 🧑‍⚕️ Patient Management (Add, Edit, Delete, Search)
- 👨‍⚕️ Doctor Management (Add, Edit, Delete)
- 📅 Appointment Booking & Cancellation
- 📊 Dashboard with Total Counts
- 🔄 REST API Integration

## 🗄️ Database Tables

- patients
- doctors
- appointments
- admin
